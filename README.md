airport-control
===============

Node.JS + MySQL панель управления аэропортом

Развертывание:

1. Создать базу данных из дампа в sql/dump.sql.
2. Прописать в webserver.js параметры соединения с базой данных.
3. Выполнить
`cd nodejs`
`npm install`
`node webserver.js`
5. Открыть index.html в современном браузере.
